# ScratchPay – Digital Scratch Card Reward System

ScratchPay is a web-based scratch card reward system that allows users to generate and scratch digital cards to reveal rewards such as cashback, vouchers, or promotional offers.

This project simulates the scratch card experience commonly used in fintech and marketing platforms like Paytm, CRED, and Google Pay.

---

## Project Overview

ScratchPay provides an interactive digital scratch card experience where users can:

1. Generate a scratch card.
2. Scratch the card surface using their mouse.
3. Reveal a hidden reward generated by the backend.
4. View generated rewards in the admin dashboard.

The project demonstrates frontend–backend integration, API communication, and interactive UI design using HTML Canvas.

## How It Works

1. User opens the ScratchPay interface.
2. User clicks **Generate Scratch Card**.
3. The frontend sends a request to the Flask backend.
4. The backend randomly selects a reward.
5. The reward is sent back to the frontend.
6. The user scratches the card to reveal the reward.

## Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/scratchpay.git
cd scratchpay
```

### 2. Install Dependencies

```bash
pip install flask
pip install flask-cors
```

### 3. Run the Backend Server

```bash
python app.py
```

The server will start at:

```
http://127.0.0.1:5000
```

### 4. Open the Frontend

Open the following file in your browser:

```
frontend/index.html
```

### 5. Open the Admin Dashboard

```
admin/dashboard.html
```
